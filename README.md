# 2D, Inviscid Compressible Euler Equations Solver

This project is a basic implementation of the 2D compressible Euler equations. The main goal is to learn and experiment with numerical schemes, the Courant–Friedrichs–Lewy (CFL) condition, and explicit time-stepping methods.

## Project Goals

- Understand and implement the 2D compressible Euler equations.
- Explore various numerical schemes for solving hyperbolic PDEs.
- Study the impact of the CFL condition on stability.
- Gain hands-on experience with explicit time integration methods.

## Current Status

The project is in its early stages. Initial code structure and basic setup are being developed.
